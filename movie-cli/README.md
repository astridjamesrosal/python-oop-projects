# Movie Command Line Interface App

## Features
- Add a movie with a title, director, year, and genre.
- View all movies in the collection.

## How To Run
- Clone the repository
- Navigate to the movie-cli folder
- Run `python main.py`

## What I learned
- Implemented a Constructor (__init__) that initializes the object's attributes when an instance is created.
- Implemented an instance method where a function is defined inside a class that operates on an object's data.
- Applied Encapsulation to keep the data and logic that acts on it inside the class that owns it. No unnecessary data for the main loop that it doesn't need to run.
